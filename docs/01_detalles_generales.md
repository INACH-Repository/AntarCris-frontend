# Documentacion del frontend CRIS Antártico
Este documento busca ser la puerta de entrada a la documentacion del sistema CRIS Antártico, 
dando una vista general de lo que hay en cada documento, para que el desarrollador o mantenedor 
tenga una idea de lo que se puede encontrar.

Dentro de esta carpeta se encuentran 4 archivos `.md` en donde se resumen los detalles mas importantes 
para levantar, desarrollar y mantener la aplicacion.

## 02_configuracion_instalacion
En este archivo se dan las instrucciones basicas para levantar el frontend del CRIS, desde software 
que se tiene que instalar hasta los comandos que se tiene que ejecutar.

La idea tener las instrucciones necesarias lo mas simple y claro posible.

## 03_componentes
En este archivo se indica el como fue construida la interfaz almacenada en este repositorio, ademas de 
algunas directrices para agregar nuevos componentes a la interfaz que puedan ser de interes.


## 04_editar_interface
En este archivo se dan las instrucciones necesarias para hacer los cambios esteticos a la interfaz web, desde 
que archivos a editar hasta como agregar nuevas imagenes.


## 05_entorno_produccion
En este archivo se dan las instrucciones necesarias para levantar la aplicacion y como esta se vincula con los 
diferentes pipelines de github actions.
